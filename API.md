## API

### POST

- `http://localhost:3000/api/expense`

### PUT :

- `http://localhost:3000/api/expense/:id`

### DELETE :

- `http://localhost:3000/api/expense/:id`

### GET

- `http://localhost:3000/api/expense`
- `http://localhost:3000/api/expense/:startDate/:endDate?`
- `http://localhost:3000/api/expense/:category/:startDate/:endDate?`
